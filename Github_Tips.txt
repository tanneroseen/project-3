Tips for using github.

Create a repository on the website invite all particapants to it 

when a file is made in it everytime before you start changing the last version type in the terminal:

git pull origin

Then edit your piece. When done type:

git add "name of the file (quotes if there's a space in the file name)"

Then type:

git commit -m "A message stating the changes you made"

Then lastly:

git push

Then your changes should be made.
